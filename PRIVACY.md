# Hermes Chrome Connector Privacy Policy

Last updated: June 23, 2026

Hermes Chrome Connector is a companion extension for Hermes Agent. It connects Google Chrome to a local Hermes Agent bridge running on the user's device.

The extension may access browser tabs, page URLs, page titles, page content, screenshots, and user interactions when the user authorizes Hermes to perform browser-based tasks.

The extension uses this information only to complete user-requested Hermes tasks through the local Hermes Agent workflow. It does not sell user data, use user data for advertising, use user data to determine creditworthiness, or use user data for lending purposes.

Hermes Chrome Connector communicates with the local Hermes Agent bridge on the user's device. The extension does not provide a standalone cloud service.

Users control when Hermes is allowed to operate Chrome. Removing the extension disables its browser access.
